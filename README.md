QR Code Generator

Esse projeto foi feito em Python pra treinar geração de QR Code, código de barras, senha aleatória e algumas automações simples com interface.

O que ele faz

- Gera QR Code a partir de texto
- Gera código de barras (EAN-13)
- Gera senhas aleatórias
- Salva as imagens geradas
- Mostra notificações no Windows
- Interface feita com Tkinter

Como rodar

Primeiro instala as dependências:

python -m pip install -r requirements.txt

Depois é só executar:

python main.py

Organização do projeto

Control → onde ficam as funções principais (QR, barras, senha, notificações)
View → parte da interface
Img → onde ficam imagens ou ícones
main.py → arquivo principal que roda tudo

Sobre

Projeto feito pra estudo de Python, automação e manipulação de arquivos/imagens.
